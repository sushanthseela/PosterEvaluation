Poster Evaluation Metrics

Logos Evaluation

The system checks the logos for size, clarity, and contrast, ensuring alt text is provided.

Font Style Evaluation

The system checks the font style for readability, ensuring clear fonts are used

Font Size Evaluation

The system evaluates the font size for different sections (title, headings, body text).

Color Contrast Evaluation

The system checks the contrast ratio between text and background.

Poster Layout Evaluation

The system evaluates the overall layout for logical structure, spacing, and flow.

Hyperlinks Evaluation

The system checks if hyperlinks are functional and accessible.
